### Setup
  1. First clone this repo to your local machine!
    - using ssh key <br>
      ```shell
      $ git clone git@github.com:kbhoffmann/battleship.git
      ```
    - - using https <br>
      ```shell
      $ git clone https://github.com/kbhoffmann/battleship.git
      ```
  2. Run `gem install bundle` in your terminal
